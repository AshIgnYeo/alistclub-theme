# CLAUDE.md

Project-specific instructions for Claude Code.

## Project context

This theme is a redesign of a very old A-List Club WordPress theme. Use the existing codebase as a reference for business logic, layout intent, and integration points (WooCommerce hooks, custom post types, search behaviour, etc.), but feel free to overhaul implementations where the old approach is outdated, insecure, or incompatible with modern WordPress / WooCommerce. Modernisation is encouraged — backwards-compatibility shims for the old theme are not required.

## Commit messages

- Do **not** add `Co-Authored-By: Claude…`, `Generated with Claude Code`, or any other AI-attribution lines to commit messages or PR descriptions.
- Keep commit messages focused on the change itself.
